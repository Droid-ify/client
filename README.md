## üì± About Droid-ify

> **Clutterfree F-Droid client**

**Droid-ify** is a modern, fast, and feature-rich **F-Droid client** for Android that brings the power of open-source app distribution to your fingertips with a beautiful Material 3 design. Built as a direct adaptation of Foxy-Droid, Droid-ify provides a smooth and intuitive way to discover, install, and manage applications from F-Droid repositories.

### What is F-Droid?
F-Droid is a catalog of libre and open-source software (FOSS) applications for Android. Droid-ify serves as an alternative client to the official F-Droid app, offering enhanced performance, modern UI, and additional features while maintaining full compatibility with F-Droid repositories.

- **Clean animations** - Smooth transitions without unnecessary visual clutter

### ‚ö° **Performance & Syncing**
- **Lightning-fast repository syncing** - Optimized network requests and parsing
- **Background auto-sync** - Keep your app catalog up-to-date automatically
- **Smart caching** - Efficient storage and retrieval of repository data
- **Multiple sync options** - WiFi-only, always, or WiFi + charging modes

### üõ† **Installation Methods**
- **Session Installer** - Modern Android package installer (recommended)
- **Legacy Installer** - Traditional installation method
- **Root Installer** - Silent installs with root access
- **Shizuku Integration** - ADB-powered installations without root

### üåê **Repository Management**
- **Multiple repositories** - Support for F-Droid, IzzyOnDroid, and custom repos
- **30+ pre-configured repositories** - Including Guardian Project, Bitwarden, KDE, and more
- **Easy repository addition** - One-click setup via repository links
- **Repository validation** - Cryptographic signature verification

### üì¶ **App Management**
- **Smart updates** - Automatic detection and installation of app updates
- **Favorites system** - Mark and organize your preferred apps
- **Detailed app information** - Comprehensive metadata, screenshots, and changelogs
- **App sharing** - Share apps via deep links
- **Search & filtering** - Find apps by name, category, or features

### üîí **Security & Privacy**
- **Signature verification** - Ensures app authenticity and integrity
- **Anti-feature warnings** - Clear indicators for ads, tracking, or proprietary components
- **Proxy support** - HTTP/SOCKS proxy for enhanced privacy
- **No tracking** - Privacy-focused design with minimal data collection
